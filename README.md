# brasil_publico

[![Instagram](https://img.shields.io/badge/@brasil__publico-Instagram-E1306C)](https://instagram.com/brasil_publico)

O **Brasil Público** surgiu a partir de uma insatisfação pessoal com a forma como consumimos notícias hoje, e eu me incluo no todo. Estamos constantemente lendo e compartilhando informações sem uma análise crítica ou sem considerar o contexto social. Pouco nos perguntamos sobre o que aqueles números realmente significam ou se estamos deixando passar algo.

Este repositório é a **camada técnica** por trás do Instagram [@brasil_publico](https://instagram.com/brasil_publico).

---

## O Projeto

A proposta não é apenas divulgar dados, mas **contrapor narrativas midiáticas com os mesmos dados**, revelando o que os números podem esconder quando lidos sem contexto. O projeto nasce do incômodo com o hábito comum de tirar conclusões sem embasamento.

Aqui, vou direto à fonte: esmiuço os dados não para forçar uma narrativa pronta ou provar um ponto, mas para encontrar a **evidência real** seja ela otimista, pessimista ou apenas factual e traduzi-la em uma linguagem simples e acessível. É fundamental começarmos a desenvolver nossa própria interpretação em um momento em que tantas informações parecem impostas.

### Princípios Norteadores

Para garantir a integridade das análises e evitar o viés de confirmação, sigo estes pilares:

- **Hipótese Primeiro:** Formular a hipótese antes de olhar os números.
- **Rastreabilidade:** Documentar cada decisão de limpeza em um arquivo `DECISIONS.md` dentro de cada análise.
- **Rigor Estatístico:** Distinguir correlação de causalidade explicitamente.
- **Honestidade Intelectual:** Comunicar incertezas de forma honesta. Se os dados forem inconclusivos, a análise dirá exatamente isso.
